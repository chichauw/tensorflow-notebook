## Add files to repositories by Git

Create a new file in local folder, for example, file.txt. 

>\> git status

>\> git add file.txt

>\> git status

>\> git commit -m "new file"

>\> git push origin master




## Remove files from repositories by Git

>\> git rm file1.txt

>\> git commit -m "remove file1"

>\> git status

>\> git push origin master
